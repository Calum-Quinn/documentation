## Full Example Projects

The GitHub repository linked below showcases a set of simple bots created using Discord4J

* [Example Projects GitHub Repository](https://www.github.com/Discord4J/example-projects)

### What do the example projects demonstrate?
* How to set up [Maven](https://maven.apache.org/) or [Gradle](https://gradle.org/) for a Discord4J bot.
* Have the bot connect to Discord's gateway.
* A simple set of commands (ping-pong and greet) to show how to create, delete, and edit slash commands with minimal work.
* How utilizing the [Spring Boot Framework](https://spring.io/) can eliminate large amounts of boilerplate.
* How to use [Logback](http://logback.qos.ch/), an [SLF4J](http://www.slf4j.org/) logger implementation to log to console.
* And more.
